    # 快速上手

    参考文档 (opens new window)

    热更新启动项目：

    {
  "scripts": {
    "docs:dev": "vuepress dev docs ----temp .temp",
  }
}

    # 默认主题配置

    参考文档 (opens new window)

    themeConfig: {
  //顶部导航
  nav: [
    { text: "首页", link: "/" },
    {
      text: "Github",
      link: "https://github.com/Taotaopersonal",
    },
  ],

  //自动根据文章标题生成侧边栏
  // 从 1.5.6 之后，主题将原有的侧边栏的中的多级标题移出，生成子侧边栏，放在页面的右侧。左边的侧边栏只有在自定义的侧边栏时才会出现，用于展示多篇文章的集合，而子侧边栏专注于当前文章的多级标题。
  sideSubbar:'auto',
  //侧边栏菜单
  sidebar: [
    {
      title: "VuePress搭建博客",
      path: "/handbook/Blog",
      collapsable: false, // 不折叠
      children: [
        {
          title: "kaishi ",
          path: "/handbook/Blog",
        },
      ],
    },
  ],
},

    # 使用其他主题

    参考文档 (opens new window)

    如果你的主题名以 vuepress-theme- 开头，你可以使用缩写来省略这个前缀：

    module.exports = {
  theme: "reco", //
};

    # github Actions

    参考文档 (opens new window) 将代码同步到 gitee,并且使用 gitee 的 pages 服务再将服务重新构建部署一次。

    module.exports = {
  title: "前端笔记",
  base: "/notes/",
  theme: "reco",
  locales: {
    "/": {
      lang: "zh-CN",
    },
  },
  description: "前端,文档,技术文档,博客搭建,vuepress",
  themeConfig: {
    nav: [
      { text: "首页", link: "/" },
      {
        text: "Github",
        link: "https://github.com/Taotaopersonal",
      },
    ],
    //   displayAllHeaders: true,
    //   sidebarDepth: 10,
    //   sidebar: "auto",
    subSidebar: "auto",
    sidebar: [
      {
        title: "欢迎学习",
        path: "/",
        collapsable: false,
        children: [{ title: "学前必读", path: "/" }],
      },
      {
        title: "技术",
        path: "/Technology/Blog",
        collapsable: false,
        children: [{ title: "vuepress搭建博客", path: "/Technology/Blog" }],
      },
      {
        title: "文章",
        path: "/Thought/Article1",
        collapsable: false,
        children: [{ title: "文章一", path: "/Thought/Article1" }],
      },
    ],
  },
};
